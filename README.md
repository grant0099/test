# 專案建置步驟

專案初始化安裝 (第一次開啟專案，安裝專案所需套件):
```sh
npm install
```

執行專案 (使專案可以在本機執行):
```sh
npm run serve
```

`**目前host domain設定為 rd3-localhost-test.opview.com.tw`  
`需要去C:\Windows\System32\drivers\etc\host設定:`  
```
127.0.0.1 rd3-localhost-test.opview.com.tw
```

專案建置: 檔案位置於`yourProject/build/Trend`
```sh
npm run build
```
